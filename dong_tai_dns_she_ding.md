# 動態DNS設定

1. 先至 https://www.dtdns.com/ 註冊帳號
2. 至 Hostnames 註冊一個子網域  
![](火狐截圖_2015-04-25T17-08-37.868Z.png)
3. 執行 Extra / DtDNS 填入子網域名稱，以及DtDNS密碼  
![](snap004.png)
4. 儲存變更（Save Changes），靜候五分鐘至十分鐘
